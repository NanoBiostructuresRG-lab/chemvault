# Curador-BD

Una aplicación de Streamlit para gestionar, visualizar y explorar bases de datos de forma interactiva.

## Requisitos

- Python 3.8+
- Streamlit
- Pandas
- NumPy

## Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/NanoBiostructuresRG-lab/Curador-BD.git
cd CURADOR-BD
```
2. Crear un entorno de conda (opcional)
```bash
conda create -n NOMBRE python=3.12
```

3. Instala las dependencias:
```bash
pip install -r requirements.txt
```
4. Correr la app
```bash
python3 -m streamlit run app.py
```
O solamente
```bash
streamlit run app.py
```

## Uso

Ejecuta la aplicación con:
```bash
streamlit run app.py
```

Luego abre tu navegador en `http://localhost:8501`
